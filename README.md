Frontend 파일에 들어가 src 파일에 들어가면 프론트 화면을 구성하기 위해 사용한 pages 파일 코드, 서버와 정보 교환을 위한 api 파일 코드, 
시스템의 전체적인 라우팅과 실행을 담당하는 App.jsx 파일 코드를 확인할 수 있습니다. 또한, App.css 파일 코드를 통해 디자인에 활용된
코드들을 확인할 수 있습니다.

Backend 파일에 들어가면 관리자 권한을 검사하는 adminMiddleware.js 파일 코드, 로그인 유무를 검사하는 authMiddleware.js 파일 코드,
DB와 프론트 사이에서 정보를 매개하여 전달하는 server.cjs 서버 파일 코드를 확인할 수 있습니다.
